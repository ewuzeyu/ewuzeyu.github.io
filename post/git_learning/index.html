git操作学习笔记 一

Posted on 2022-02-19，2 min read

git操作的学习笔记 一

链接
git操作

git 工作流程

  • 工作区(Workspace)
  • 暂存区(Index/Stage)
  • 本地仓库(Respository)
  • 远程仓库(Remote)

远程仓库<-(push)- -(pull/clone)->本地仓库<-(commit)-暂存区<-(add)-工作区

基本工作流程

  1. 在工作区中添加修改文件
  2. 工作区中的文件放入暂存区
  3. 暂存区的文件提交本地git本地仓库
  4. 本地仓库推送到远程仓库

操作

初始化

两种方式初始化

# 方式一:本地生成git
git init
# 方式二:远端克隆仓库
git clone https://github.com/xx/xx.git

基本配置

# 配置用户名
git config --global user.name "name"
# 配置邮箱
git config --global user.email "name@mail.com"

删除远程仓库

git remote rm origin

添加远程仓库

git remote add origin https://github.com/xx/xx.git

推送到远程仓库

将已修改文件添加到暂存区

git add dir/filename # 添加指定文件
git add . # 添加所有已修改文件

暂存区改动提交到本地仓库

git commit命令后在本地仓库生成40位哈希值,用于版本回退

git commit -m "message" # message即本次提交的说明

上传至远程仓库

git push -u origin master # master可以换为其他分支

从远程仓库拉取

更新本地

git pull origin master # master可以换为其他分支
